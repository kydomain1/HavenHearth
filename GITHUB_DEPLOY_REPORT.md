## 🚀 本地开发

### 克隆仓库
```bash
git clone https://github.com/kydomain1/HavenHearth.git
cd HavenHearth
```

### 运行网站
方法1：使用Live Server（VS Code扩展）
- 右键 `index.html`
- 选择 "Open with Live Server"

方法2：使用Python
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

方法3：使用Node.js
```bash
npx http-server
```

然后访问 `http://localhost:8000`
