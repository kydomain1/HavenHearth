## 🎯 功能特点

### 1. **智能搜索算法** 🧠

搜索系统使用**评分机制**来确定相关性：

```
标题匹配 = 10分
关键词匹配 = 8分
分类匹配 = 7分
摘要匹配 = 5分
```

结果按相关性得分排序，最相关的排在前面。

### 2. **多字段搜索** 🔍

搜索会检查：
- ✅ 文章标题
- ✅ 文章摘要
- ✅ 文章分类
- ✅ 关键词标签

**示例**：
- 搜索"coffee" → 找到咖啡文章
- 搜索"fashion" → 找到所有时尚类文章
- 搜索"wellness" → 找到健康相关文章

### 3. **分类过滤** 🎯

搜索结果可以按分类筛选：
- All Categories（全部）
- Fashion（时尚）
- Health（健康）
- Home（家居）
- Travel（旅游）
- Food（食品）

### 4. **URL参数支持** 🔗

搜索使用URL参数，支持：
- 📋 直接访问：`search.html?q=coffee`
- 🔄 刷新页面保持搜索
- 🔗 分享搜索结果链接
- ⬅️ 浏览器前进/后退

### 5. **友好的无结果处理** 💡

当没有找到结果时显示：
- 😊 友好的提示消息
- 💡 搜索建议
- 🔗 热门话题链接
- 🏠 返回主页按钮

### 6. **加载动画** ⏳

- 搜索时显示旋转加载图标
- "Searching articles..." 提示
- 平滑的结果显示动画

## 🚀 使用方法

### 方式1：从任何页面搜索
1. 在任何页面的搜索框输入关键词
2. 点击搜索按钮或按Enter
3. 自动跳转到 `search.html?q=关键词`
4. 显示搜索结果

### 方式2：直接访问搜索页
1. 访问 `search.html`
2. 在大搜索框中输入关键词
3. 查看结果

### 方式3：URL直接搜索
1. 直接访问：`search.html?q=你的搜索词`
2. 页面自动显示结果

### 方式4：点击热门话题
1. 在无结果页面
2. 点击"Popular Topics"下的链接
3. 快速搜索热门主题

## 🎯 文章数据库

```javascript

- readTime: 阅读时间
```

## 🔧 技术实现

### 搜索算法流程

```
1. 获取URL参数中的搜索词
   ↓
2. 转换为小写并去除空格
   ↓
3. 遍历文章数据库
   ↓
4. 对每篇文章评分：
   - 标题匹配 +10
   - 关键词匹配 +8
   - 分类匹配 +7
   - 摘要匹配 +5
   ↓
5. 按得分排序（高到低）
   ↓
6. 生成结果HTML
   ↓
7. 显示在页面上
```

### 分类过滤流程

```
1. 点击分类按钮
   ↓
2. 更新按钮active状态
   ↓
3. 遍历所有文章卡片
   ↓
4. 检查分类匹配
   ↓
5. 显示/隐藏对应文章
   ↓
6. 更新结果计数
```
